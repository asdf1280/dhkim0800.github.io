ETS2 Auto Save Editor - Applying a shared skin

ETS2 자동 세이브 편집기(ETS2ASE)를 이용하여 다른 사람이 공유한 스킨을 내 트럭/트레일러에 쉽게 적용하는 방법에 대하여 알아 봅시다. 먼저 ETS2ASE를 설치하세요. 이 링크로 이동하여 설치 방법을 확인하세요.

방법 알아보기

다른 사람이 공유한 스킨 파일을 준비하고, 파일의 위치를 미리 확인하십시오. 파일의 확장자는 .paint0(트럭)과 .paint1(트레일러) 입니다. 일반적으로는 트럭과 트레일러 스킨을 동시에 배포하니, 두 개의 파일이 있으면 됩니다. (트럭과 트레일러를 개별로 하나씩 배포할 수도 있습니다.)

주의

스킨 파일을 직접 열지 마세요! 스킨 파일은 세이브 편집기가 자동으로 해석하여 작동하니, 임의로 수정하여 발생하는 결과에 대해 책임지지 않습니다.

트럭/트레일러 할당

이 편집 기능은 ETS2 프로필 내에서 플레이어에게 할당된 트럭/트레일러를 기반으로 작동합니다. 만약 할당된 트럭이 없다면, 트럭 스킨을 적용할 수 없고, 트레일러가 없다면 트레일러 스킨을 적용할 수 없습니다. 미리 ETS2를 실행하여 플레이어에게 트럭을 지정한 뒤 트레일러를 연결하시기 바랍니다.

ETS2ASE 프로그램을 실행합니다. 이후 아래 사진과 같이 프로필, 세이브파일 선택한 뒤, 작업 목록에서 '페인트 내보내기/불러오기'를 선택합니다. 사진과 같이 '시작!'을 누릅니다.

시작! 버튼을 누르면 프로그램은 트레일러와 트럭, 불러오기와 내보내기를 질문하는 창을 표시할 것입니다. 지금 해야 할 일은 스킨을 불러오는 것이기 때문에 불러오기를 선택합니다.

트럭과 트레일러의 스킨 중 불러오고자 하는 것을 선택합니다. 트럭, 불러오기 또는 트레일러, 불러오기를 선택한 뒤 확인을 누릅니다. 파일을 선택하는 창이 표시됩니다. 미리 준비해둔 스킨 파일이 위치한 곳에 이동한 뒤, 해당 스킨 파일을 선택하세요. '트럭, 불러오기'를 선택했다면 트럭용 스킨만 표시되고, '트레일러, 불러오기'를 선택했다면 트레일러에 사용할 수 있는 스킨만 표시됩니다. 파일을 선택한 뒤 '확인'을 누르면 자동으로 해당 스킨을 적용합니다. 이제 ETS2를 실행하여 변경 사항을 확인하시기 바랍니다.
